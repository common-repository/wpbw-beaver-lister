Showing builder enabled status and making the list of the used modules

== Description ==

Plugin is checking that a page, post or cpt is using the page builder or not. If page builder is activated on a page, post or cpt, it will make a list of used modules.

The plugin only works with [Beaver Builder](https://goo.gl/3kNSy8 "Beaver Builder") plugin.

Here is a video explaining how plugin is working

[youtube https://youtu.be/7LbEXB_D9Og]

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wpbw-beaver-lister` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Dashboard->Pages->All Pages and see the two new columns: Builder Enabled and Modules Used
